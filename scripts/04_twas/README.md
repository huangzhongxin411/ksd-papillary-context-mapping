# 04 TWAS

## Goal

Prioritize genes whose genetically regulated expression is associated with KSD risk.

## Candidate Tools

- FUSION TWAS.
- S-PrediXcan / S-MultiXcan.

## Tissue Strategy

Use kidney cortex where available, but include cross-tissue models because KSD risk may involve renal, metabolic, inflammatory, and mineral-handling systems.

## Output

- Significant and suggestive TWAS genes.
- Tissue distribution table.
- Locus-level TWAS summaries.
