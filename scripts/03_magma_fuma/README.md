# 03 MAGMA / FUMA

## Goal

Run gene-based, gene-set, tissue, and cell-type enrichment analyses.

## Main Outputs

- MAGMA gene-based table.
- GO / KEGG / Reactome / MSigDB enrichment.
- Kidney tissue and renal papilla cell-type enrichment summaries.

## Notes

The 2023 KSD GWAS already reported proximal tubule relevance, so this project should go beyond repeating that result and focus on renal papillary epithelial, transport, immune, and stromal compartments.
