# 07 snRNA Gene Mapping

## Goal

Map Tier 1/2/3 candidate genes and pathway modules onto renal papillary cell types.

## Analyses

- Candidate gene dot plots across annotated cell types.
- Donor x cell type pseudobulk expression.
- Module scores for Tier 1, Tier 1+2, and pathway-specific gene sets.
- Random expression-matched gene set benchmarks.

- `results/tables/scrna_candidate_gene_localization.tsv`
- `results/tables/scrna_module_scores.tsv`
- Cell-type localization figures for Figure 3.
