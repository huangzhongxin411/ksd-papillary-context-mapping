# Lightweight v3.3 Submission-Package Index

This directory is a lightweight public submission-package index for manuscript v3.3 and the future human-approved GitHub Release v1.0.1. Authoritative files remain in their canonical repository directories. Large manuscript, figure, supplementary-figure and supplementary-table payloads are intentionally not duplicated here, avoiding unnecessary public payload and version drift.

## Canonical locations

- Manuscript files: `manuscript/`
- Main figures: `figures/`
- Supplementary figures: `supplementary_figures/`
- Supplementary tables: `supplementary_tables/`
- Supplementary legends and captions: `supplementary_materials/`
- Source-data manifest: `source_data/Source_Data_manifest_v3.3.tsv`
- Repository manifest: `MANIFEST.tsv`
- Repository checksums: `CHECKSUMS.sha256`

`Submission_file_manifest_v3.3.tsv` lists submission-facing canonical paths. `Submission_package_mapping_v3.3.tsv` explains how package components map to those paths, and `BMC_Genomics_readiness_checklist_v3.3.tsv` separates completed technical checks from human-only requirements.

The repository Zenodo DOI is pending and must not be invented. GitHub Release v1.0.0 remains the pre-Zenodo checkpoint. After final human approval, the manuscript-synchronized archive should be minted from GitHub Release v1.0.1 and its Zenodo DOI should then be backfilled into the relevant manuscript and repository metadata.
