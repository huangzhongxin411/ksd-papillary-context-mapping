# Data availability

The Cao et al. kidney stone disease GWAS summary statistics are available from Zenodo at https://doi.org/10.5281/zenodo.14790324. The associated GWAS Catalog study record is GCST90652506.

Public transcriptomic data are available from the Gene Expression Omnibus under GSE231569, GSE73680, GSE206306 and GSE231630. GTEx v8 / PredictDB Kidney_Cortex MASHR prediction models are available from their original distribution resource and remain subject to the applicable terms.

This repository contains analysis code, author-generated result tables, figure source data, supplementary tables, manifests and selected release-support materials: https://github.com/huangzhongxin411/ksd-papillary-context-mapping

It does not redistribute identifiable human data, large public raw datasets, 1000 Genomes LD reference files, GTEx/PredictDB models or other restricted third-party resources.

The current manuscript-synchronized repository release will be v1.0.1 after final human approval. GitHub Release v1.0.0 is retained as a pre-Zenodo checkpoint. The repository archive citation and DOI will be added only after the final release is archived and a Zenodo DOI is minted.
